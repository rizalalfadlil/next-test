Dokumentasi aplikasi

untuk memulai, buka halaman /login , terdapat 3 jenis akun yang dapat digunakan dengan yaitu :

tipe akun kelola daftar user lihat riwayat aktivitas kelola daftar menu membuat pesanan baru
admin ✅ ✅
manajer ✅ ✅
kasir ✅

Menggunakan aplikasi sebagai admin

  • kelola daftar user dengan membuka /users lalu :
    • klik tombol untuk menambah akun baru
    • klik tombol di salah satu record untuk menghapus sebuah akun
    • klik tombol untuk melihat atau mengedit akun
  • lihat riwayat aktivitas dengan membuka halaman /activity

Menggunakan aplikasi sebagai manajer

  • kelola daftar menu dengan membuka halaman /menu
    • buat menu baru dengan menekan tombol tambah data
    • edit sebuah data dengan menekan tombol edit di salah satu menu yang ada atau hapus dengan menekan tombol hapus
  • lihat riwayat aktivitas dengan membuka halaman /activity
  • lihat riwayat pemesanan dengan membuka halaman /order
    • lihat salah satu detail pesanan lebih lengkap dengan menekan tombol lihat di salah satu datanya

Menggunakan aplikasi sebagai kasir

  • lihat riwayat pemesanan dengan membuka halaman /order
    • lihat salah satu detail pesanan lebih lengkap dengan menekan tombol lihat di salah satu datanya
    • buat pesanan baru dengan menekan tombol buat pesanan baru
